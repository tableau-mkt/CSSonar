> Detect CSS selector prevalence in your web application.


## Install

```sh
$ npm install --save cssonar
```


## Usage

```js
var cssonar = require('cssonar');

cssonar('Rainbow');
```

```sh
$ npm install --global cssonar
$ cssonar --help
```
